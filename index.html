Основы CSS

Для визуального оформления веб-страницы создан язык CSS. Именно этот язык отвечает за то, как наши HTML-элементы будут выведены пользователю в браузере. Научимся подключать CSS файлы и познакомимся с базовыми типами селекторов. Также изучим что такое каскадность, и как она работает в CSS.
